# API de Análisis Educativo con IA (Gemini)

Esta es una API RESTful construida con Django y Django REST Framework (DRF) que se integra con el modelo de Inteligencia Artificial de Google Gemini. Su propósito es analizar datos de estudiantes con bajo rendimiento y generar de forma estructurada (JSON) un diagnóstico psicopedagógico y actividades de aprendizaje personalizadas.

## 🛠️ Tecnologías Utilizadas
- Python 3.x
- Django & Django REST Framework
- Google GenAI SDK (`google-genai`)
- Pydantic (Para estructuración de datos de salida)
- Python-dotenv (Para manejo de variables de entorno)
